## Neural network with SoftMax regression
![[Pasted image 20260612162443.png]]

What is unique about the SoftMax function is that in logistic regression the output of a particular layer just depended on the input of that layer, unrelated to other layers.

But in case of SoftMax, every output depends on all other outputs as well.
