Why are we calling spanning Tree a tree -
1. It forms a tree like structure if try to re arrange it a little as it has no cycles.
2. It has n-1 edges as well like a binary tree
3. Every graph can have many spanning trees
4. Every graph can have multiple minimum spanning trees as well depending on the edges and weight of the edges.

## 3.1 Minimum spanning tree -

A tree having least sum of weights/ cost is called the minimum spanning tree of a particular graph.
Minimum cost spanning tree is a minimum spanning tree.
